# 耐张线夹区域分割 — 传统 CV 尝试过程记录

## 任务目标

将耐张线夹射线检测（RT）图像沿水平方向自动分割为三个区域：
- **区域1**：钢芯锚固区（最亮、最窄）
- **区域2**：铝压接区（中等亮度）
- **区域3**：铝绞线区（最宽、有股线纹理）

## 已确认的关键信息

- 耐张线夹**横向排布**，在图像中为一条水平窄带（约 250~500px 高）
- 图像中夹线上方存在同等亮度甚至更亮的**干扰物**
- 三区域沿**水平方向**排列（R1 ↔ R2 ↔ R3）
- 区域1 可以在夹线的最左端或最右端（夹线可 180° 翻转）
- 标注数据：`Data/Train/` 下 7 张 PNG + JSON（LabelMe 多边形标注格式）

## 尝试方案及发现

### 第一轮：全图垂直投影法

**思路**：对整个图像做垂直方向投影（行均值），沿垂直方向检测密度突变。

**结果**：❌ 完全失败。原因：
- 错误地将图像上方干扰物当作夹线的一部分
- 区域边界检测错误（把图像边缘误认为区域边界）
- 夹线是水平排布，不是垂直排布

### 第二轮：全图水平投影法

**思路**：对全图做水平方向投影（列均值），沿水平方向检测密度突变。

**结果**：❌ 失败。原因：
- 全图的列投影包含了大量夹线外的干扰信息
- 没有先做夹线 ROI 定位

### 第三轮：先定位夹线 ROI，再分割

**思路**：
1. 行投影找夹线水平条带
2. 在夹线 ROI 内做列投影 → 找谷底 → 区域边界
3. 按内容（亮度+纹理+宽度）自动标注区域1/2/3

**夹线定位尝试了多种方法**：

#### 3a. 阈值法（全局均值 × 0.8）
- 问题：阈值太低，包含了上方干扰物（干扰物和夹线亮度相似）
- 结果：夹线 y=[0, ~1400]，包含大量干扰

#### 3b. 峰值扩展法（从最亮行向上下扩展）
- 问题：全局行投影峰值在 y=1449，但夹线实际在 y=1491~1758
- 根本原因：**夹线并不是图像中最亮的物体**，上方干扰更亮
- 结果：夹线 y=[0, ~1800] 或 [1104, 1692]，始终偏上

#### 3c. 峰值+最近谷底法
- 从峰值行向上/下找最近的显著谷底（局部极小值）
- 问题：上方有多层亮带和暗谷，最近谷底可能不在夹线边界
- 结果：夹线 y=[1104, 1692]，上方边界偏高约 400px

#### 3d. 下半部搜索法（只在图像 40%~100% 范围内搜索）
- 问题：40% 起始位置（y=921）仍然在夹线上方干扰物范围内
- 结果：夹线 y=[1205, 1595]，仍然偏差

### 列投影边界检测分析

**在 GT 正确夹线 ROI 内测试列投影**：

| 发现 | 详情 |
|------|------|
| R1/R2 边界 | 有**极浅**谷底（深度仅 2.2~2.9% 投影范围），在 x≈530 处，sigma≤15 才能检测到 |
| R2/R3 边界 | **无谷底** — 铝压接→铝绞线是渐变纹理过渡，非密度突变 |
| 唯一深谷 | 在夹线右边缘（x≈2290），是金属→空气的过渡，非内部边界 |

列投影在正确夹线 ROI 内的表现（sigma=10）：

```
Column proj within clamp (GT y=1491-1758):
Valleys:  x=2291 (12.2% depth, right edge)
          x=527  (2.6% depth, near GT R1/R2 boundary ~490)
```

**GT 区域边界**: R1/R2≈490, R2/R3≈1540  
**检测到的谷**: 仅 x≈530（对应 R1/R2），x≈1540 处无谷

## 核心问题总结

### CV 方案的根本性困难

1. **夹线非最亮**：图像中夹线上方存在同等或更亮的干扰物（可能为其他金属部件），行投影峰值不在夹线内，导致基于亮度的定位策略全部失效

2. **干扰不可预测**：不是每张图都只有耐张线夹，干扰物的位置、亮度、数量因图而异

3. **R2/R3 无密度边界**：铝压接区和铝绞线区之间的过渡是**纹理差异**（有无股线纹理），而非密度差异，列投影无法检测此边界

4. **浅谷易被噪声淹没**：即使 R1/R2 的密度边界也极浅（2~3% 投影范围），对平滑参数和阈值极度敏感

### 代码产出

尽管 CV 方案不够鲁棒，以下模块仍有参考价值：
- `DL/segmentation/projection.py` — 投影分析、谷底检测、纹理分析
- `DL/segmentation/rotate.py` — 方向检测、旋转校正
- `DL/segmentation/visualize.py` — 分割结果可视化
- `DL/segmentation/region_splitter.py` — 区域分割主模块（含按内容标注逻辑）
- `DL/segmentation/batch_run.py` — 批量处理脚本

### 标注数据分析结论

从 7 张标注数据中提取的统计规律：
- 区域1（钢芯锚固）：最窄（327~580px，~18% 夹线宽）、最亮
- 区域2（铝压接）：中等（769~1047px）
- 区域3（铝绞线）：最宽（713~1298px）、纹理最强
- 区域1 可以在夹线任一端（翻转 180°）
- 有一张图只有 2 个区域（无区域3）

## 后续方向

**推荐采用深度学习方案**：
1. **夹线检测**：YOLOv8/轻量检测模型，定位夹线 bbox
2. **区域分割**：在夹线 ROI 内，用 1D-CNN 沿水平方向滑动窗口分类 R1/R2/R3，或 U-Net 语义分割
3. 7 张标注数据 + 强数据增强可初步训练，后续 200+ 张数据逐步迭代
